# Rybbit on Eyevinn Open Source Cloud

[Rybbit](https://github.com/rybbit-io/rybbit) is a privacy-friendly, open-source Google Analytics alternative featuring session replay, real-time analytics, funnel analysis, error tracking, and Web Vitals monitoring.

This repository has been adapted for deployment on [Eyevinn Open Source Cloud (OSC)](https://www.osaas.io) — a platform that makes it easy to run open source software as a managed service.

## Deploy on OSC

With Eyevinn Open Source Cloud, you can deploy a fully managed Rybbit instance in minutes. No server setup, no Docker expertise required.

1. Sign up at [app.osaas.io](https://app.osaas.io)
2. Search for "Rybbit" in the service catalog
3. Click "Create new instance" and fill in your configuration
4. Your analytics platform is live

## What OSC Provides

- Automatic scaling and high availability
- Managed PostgreSQL, ClickHouse, and Redis
- SSL/TLS termination and custom domains
- Monitoring and alerting
- Automatic backups
